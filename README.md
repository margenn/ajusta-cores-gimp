# ajusta-cores-gimp
Plugin em python para ajustar cores de várias imagens em uma determinada pasta

## COMO UTILIZAR

Coloque o script na pasta ~/.config/GIMP/3.0/plug-ins/ajusta-cores//ajusta-cores.py

Reinicie o GIMP

Depois vá em Filters > "DRONE AJUSTA CORES"

Escolha as pastas de entrada (onde estão os arquivos originais) e a de saída (onde os arquivos editados serão salvos)

Confirme e aguarde o processamento


## Limitações e bugs

Este script foi testado com o Gimp na versão 3.0 em Linux.
